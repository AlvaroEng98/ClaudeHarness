---
name: plan-with-agent
description: Genera spec.md y Plans.md para la tarea actual usando el agente Planificador.
---

Invoca al agente `planificador` con el request del Desarrollador como tarea.

El planificador explorará el codebase y producirá dos artefactos en la raíz del proyecto:
- `spec.md` — descripción del problema y decisiones de diseño.
- `Plans.md` — lista ordenada de tasks para el Constructor.
